The Harry Potter API

Task:

By using a publicly available dataset, or one of your own making, we would like you to develop an application showcasing your development capabilities. The application functionality can be anything you wish to develop but should include data retrieval, manipulation and presentation.

My Application:

In this application, users can search for characters from the Harry Potter novels and have information relayed back to them in a clear but aesthetically pleasing way.
To make this happen I used the Harry Potter API. This API returns JSON data holding information about characters such as their Name, Hogwarts House, Species and more.
Through the use of the input textbox, users can search for their favourite characters. By clicking the 'submit' button the textbox content it adding to a skeleton URL and JSON data is then returned and printed in a readable manner. They can also choose to have a random character selected for them.
Also, see how the card background changes based on the character's Hogwarts House!
